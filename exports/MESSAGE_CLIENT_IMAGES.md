# Message à envoyer au client - Clarification sur les images

---

**Objet :** Clarification : Association des images aux variations

Bonjour,

J'ai compris que les "références" mentionnées dans vos instructions (ex: JOJO1015-24, JOJO1015-25) correspondent en fait aux **noms des images disponibles**, et non à un attribut "Référence" dans WooCommerce.

## Situation actuelle

Pour la collection **"Coque Luxury Transparente"**, vous avez indiqué :
- **2 images disponibles** : JOJO1015-24.JPG et JOJO1015-25.JPG
- **7 couleurs** : Argent, Rose, Bleu, Violet, Noir, Or, Rouge
- **4 modèles** : iPhone 17, iPhone 17 Air, iPhone 17 Pro, iPhone 17 Pro Max

Cela donne **28 variations** (4 modèles × 7 couleurs).

## Question principale

**Comment associer les 2 images aux 28 variations ?**

### Option A : Toutes les variations utilisent la même image
- Toutes les 28 variations utilisent **JOJO1015-24.JPG** (ou JOJO1015-25.JPG)
- Les 2 images sont dans la **galerie du produit parent**

### Option B : Les images sont associées selon le modèle
- Exemple : iPhone 17 et iPhone 17 Air → JOJO1015-24.JPG
- Exemple : iPhone 17 Pro et iPhone 17 Pro Max → JOJO1015-25.JPG
- Ou une autre répartition selon le modèle

### Option C : Les images sont associées selon la couleur
- Exemple : Certaines couleurs → JOJO1015-24.JPG
- Exemple : D'autres couleurs → JOJO1015-25.JPG
- Ou une autre répartition selon la couleur

### Option D : Les images sont dans la galerie du produit parent
- Les 2 images (JOJO1015-24.JPG et JOJO1015-25.JPG) sont dans la galerie du produit parent
- Les variations n'ont pas d'image spécifique
- WooCommerce affiche les images de la galerie pour toutes les variations

### Option E : Autre logique
Si vous avez une autre méthode d'association, merci de la détailler.
